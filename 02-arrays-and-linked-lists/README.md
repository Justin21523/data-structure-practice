# 第二章：陣列與鏈結串列（Arrays and Linked Lists）

本章介紹兩種最基礎的線性資料結構：陣列（array）與鏈結串列（linked list）。它們是許多進階資料結構的基礎建構區塊。

## 學習目標

- 理解陣列與鏈結串列的記憶體配置差異
- 掌握各種串列變體的實作與操作
- 學會根據使用情境選擇適當的資料結構

## 重點比較

### 陣列 vs 鏈結串列

| 特性 | 陣列（Array） | 鏈結串列（Linked List） |
|------|--------------|------------------------|
| 記憶體配置 | 連續（contiguous） | 分散（scattered） |
| 隨機存取 | O(1) | O(n) |
| 頭部插入 | O(n) | O(1) |
| 尾部插入 | O(1) 攤銷* | O(1)** |
| 中間插入 | O(n) | O(1)*** |
| 空間額外開銷 | 低 | 每節點需存指標 |
| 快取效能 | 優（cache-friendly） | 差（cache-unfriendly） |

\* 動態陣列
\** 需維護尾指標
\*** 已知插入位置時

### 鏈結串列變體比較

| 類型 | 特色 | 額外空間 | 適用情境 |
|------|------|---------|---------|
| 單向 | 只能向前遍歷 | 1 個指標/節點 | 堆疊實作、簡單佇列 |
| 雙向 | 可前後遍歷 | 2 個指標/節點 | LRU Cache、文字編輯器 |
| 環狀 | 無終點 | 同上 | 輪詢排程、約瑟夫問題 |

## 考試常見題型

1. **陣列操作**
   - 反轉陣列（in-place）
   - 旋轉陣列
   - 合併已排序陣列

2. **鏈結串列操作**
   - 反轉鏈結串列
   - 偵測環（Floyd's cycle detection）
   - 找中間節點（快慢指標）
   - 合併兩個已排序串列

3. **綜合應用**
   - 用陣列實作串列
   - 用串列實作堆疊/佇列
